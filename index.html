Know When to Fold 'Em (prototype)

This is a prototype I made for the final project of Concept Development at Madison College, Spring 2021. The goal was to concept an educational game. I (thanks to one of many fabulous suggestions from Paul) chose to build a game that teaches the player how to fold various pieces of laundry.

As you can see, it is far from complete!

I started the project with almost no Unity (the game engine used here) experience, having only made one 2D game jam game in it a few years ago, so getting up to speed on making a 3D Unity game took some time. I also have no Blender experience, so making the few very simple models you see here took quite a while. Also, the cloth simulation module that ships with Unity does not do a few crucial things that I was banking on it doing! Sooooo, this is a very, very rough prototype.

All that said, I think it was a good learning experience, and I think if you watch the embedded video here you can get something of a sense of what the complete game might look like. I doubt I will continue working on this after turning in the assignment, but who knows! If I get a strong response that it deserves getting turned into a real game, I'll think about it.

-Casey, May 2021
